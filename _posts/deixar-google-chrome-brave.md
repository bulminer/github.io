O **Google Chrome** é o browser mais usado do mundo, mas **envia dados sobre os teus hábitos de navegação para o Google**, que os usa para anúncios personalizados. Felizmente, existem alternativas focadas na privacidade: [Brave](https://brave.com/) (EUA, mas com políticas de privacidade fortes) e [Firefox](https://www.mozilla.org/firefox/) (EUA, mas open-source e com sede na Europa).

---

## Porque Deves Migrar?
- **Privacidade:** Brave bloqueia rastreio e anúncios por defeito. Firefox tem proteções avançadas contra rastreio.
- **Sem dados para o Google:** Chrome envia dados de utilização para o Google; Brave e Firefox não.
- **Open-source:** Firefox é totalmente open-source e auditável.
- **Velocidade:** Brave é mais rápido que o Chrome porque bloqueia anúncios.

## Como Migrar do Google Chrome
1. **Descobre o Brave:**
   - Descarrega o Brave para [Windows](https://brave.com/download/), [Mac](https://brave.com/download/), ou [Android](https://play.google.com/store/apps/details?id=com.brave.browser)/[iOS](https://apps.apple.com/app/brave-private-web-browser/id1052879175).
2. **Importa os teus dados:**
   - Ao abrir o Brave pela primeira vez, seleciona "Importar dados do Chrome" (favoritos, histórico, passwords).
3. **Define o Brave como browser padrão:**
   - No Brave, vai a Definições > Definir como padrão.
4. **Experimenta o Firefox:**
   - Descarrega o Firefox em [mozilla.org/firefox](https://www.mozilla.org/firefox/).
   - Ativa as proteções de privacidade: Vai a Definições > Privacidade e Segurança > Seleciona "Estrito".
5. **Apaga os dados do Chrome:**
   - No Chrome, vai a Definições > Privacidade e segurança > Limpar dados de navegação.

---

## Dicas para Navegação Privada
- No Brave, ativa o [Brave Shields](https://brave.com/privacy-features/) para bloquear rastreio.
- No Firefox, usa extensões como [HTTPS Everywhere](https://addons.mozilla.org/firefox/addon/https-everywhere/) e [Privacy Badger](https://addons.mozilla.org/firefox/addon/privacy-badger17/).
- Experimenta o [LibreWolf](https://librewolf.net/) para uma versão do Firefox ainda mais privada.

---

## Conclusão
O Google Chrome é rápido, mas **compromete a tua privacidade**. Brave e Firefox oferecem a mesma velocidade e funcionalidades, mas sem rastreio. **[Muda hoje e navega com mais segurança!](https://brave.com/)**
