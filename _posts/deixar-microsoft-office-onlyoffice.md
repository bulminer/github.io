<h1>Deixa o Microsoft Office e Usa o OnlyOffice ou LibreOffice</h1>
<p>O <strong>Microsoft Office</strong> é o padrão para edição de documentos, mas <strong>é caro, fechado e sujeito a vigilância americana</strong> (através da Microsoft). Felizmente, a Europa tem alternativas poderosas: <a href="https://www.onlyoffice.com/">OnlyOffice</a> (França) e <a href="https://www.libreoffice.org/">LibreOffice</a> (Alemanha).</p>

<h2>Porque Deves Migrar?</h2>
<ul>
  <li><strong>Grátis:</strong> Tanto OnlyOffice como LibreOffice têm versões gratuitas.</li>
  <li><strong>Open-source:</strong> LibreOffice é 100% open-source e auditável.</li>
  <li><strong>Privacidade:</strong> Os teus documentos não são enviados para servidores americanos.</li>
  <li><strong>Compatibilidade:</strong> Ambos suportam formatos do Microsoft Office (.docx, .xlsx, .pptx).</li>
</ul>

<h2>Como Migrar do Microsoft Office</h2>
<ol>
  <li><strong>Descobre o LibreOffice:</strong>
    <ul>
      <li>Descarrega o LibreOffice para <a href="https://www.libreoffice.org/download/download-libreoffice/">Windows</a>, <a href="https://www.libreoffice.org/download/download-libreoffice/">Mac</a> ou <a href="https://www.libreoffice.org/download/download-libreoffice/">Linux</a>.</li>
      <li>Instala e abre os teus documentos do Office (é compatível com .docx, .xlsx, etc.).</li>
    </ul>
  </li>
  <li><strong>Experimenta o OnlyOffice:</strong>
    <ul>
      <li>Descarrega o OnlyOffice Desktop para <a href="https://www.onlyoffice.com/download-desktop.aspx">Windows/Mac/Linux</a>.</li>
      <li>Ou usa a versão online em <a href="https://www.onlyoffice.com/">onlyoffice.com</a>.</li>
    </ul>
  </li>
  <li><strong>Integra com Nextcloud (opcional):</strong>
    <ul>
      <li>Se usas Nextcloud, podes instalar o <a href="https://nextcloud.com/collabora/">Collabora Online</a> para edição colaborativa.</li>
    </ul>
  </li>
  <li><strong>Guarda os teus documentos na cloud europeia:</strong>
    <ul>
      <li>Usa <a href="https://nextcloud.com/">Nextcloud</a> ou <a href="https://www.infomaniak.com/en/kdrive">kDrive</a> para armazenar documentos.</li>
    </ul>
  </li>
</ol>
