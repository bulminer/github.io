<h1>Deixa o WhatsApp e Usa o Signal ou Session</h1>
<p>O <strong>WhatsApp</strong> é o mensageiro mais popular do mundo, mas <strong>pertence ao Facebook (Meta) e partilha dados com a empresa</strong>. Além disso, as mensagens são encriptadas, mas os <a href="https://www.theverge.com/2021/1/12/22227077/whatsapp-facebook-data-sharing-privacy-policy-update-backlash">metadados</a> (quem fala com quem, quando e onde) são armazenados e podem ser acessados por terceiros. Felizmente, existem alternativas europeias (ou focadas na privacidade) como <a href="https://signal.org/">Signal</a> (EUA, mas com políticas fortes de privacidade) e <a href="https://getsession.org/">Session</a> (descentralizado).</p>

<h2>Porque Deves Migrar?</h2>
<ul>
  <li><strong>Privacidade:</strong> Signal e Session não recolhem metadados (ao contrário do WhatsApp).</li>
  <li><strong>Sem anúncios:</strong> Nenhum dos dois mostra anúncios ou partilha dados com terceiros.</li>
  <li><strong>Encriptação de ponta a ponta:</strong> Todas as mensagens são encriptadas, incluindo chamadas.</li>
  <li><strong>Código aberto:</strong> Ambos são open-source e auditáveis.</li>
</ul>

<h2>Como Migrar do WhatsApp</h2>
<ol>
  <li><strong>Instala o Signal ou Session:</strong>
    <ul>
      <li><a href="https://signal.org/download/">Signal</a> (Android/iOS/Desktop).</li>
      <li><a href="https://getsession.org/download">Session</a> (Android/iOS/Desktop).</li>
    </ul>
  </li>
  <li><strong>Convida os teus contactos:</strong>
    <ul>
      <li>No Signal, usa a opção "Convite por SMS" para enviar links aos teus amigos.</li>
      <li>No Session, partilha o teu ID (não precisa de número de telemóvel).</li>
    </ul>
  </li>
  <li><strong>Exporta as tuas conversas do WhatsApp (opcional):</strong>
    <ul>
      <li>No WhatsApp, vai a Definições > Conversas > Exportar conversa (envia por email).</li>
      <li>Nota: Não podes importar diretamente para o Signal/Session, mas podes guardar como backup.</li>
    </ul>
  </li>
  <li><strong>Desativa o WhatsApp:</strong>
    <ul>
      <li>Depois de migrares, podes desinstalar o WhatsApp ou <a href="https://faq.whatsapp.com/26000030/">apagar a tua conta</a>.</li>
    </ul>
  </li>
</ol>

<h2>Dicas para Grupos</h2>
<ul>
  <li>No Signal, podes criar <a href="https://support.signal.org/hc/en-us/articles/360002282251-Groups">grupos</a> com até 1000 membros.</li>
  <li>No Element (Matrix), podes criar <a href="https://element.io/rooms">salas públicas ou privadas</a> para comunidades.</li>
</ul>

<h2>Conclusão</h2>
<p>O WhatsApp é conveniente, mas <strong>a tua privacidade não deve ser negociável</strong>. Signal e Session oferecem mensagens seguras sem comprometer os teus dados. <strong>Migra hoje e protege as tuas conversas!</strong></p>
